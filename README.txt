项目内容：这是2021腾讯游戏客户端公开课第八次作业，也是第九次课AI入门布置的作业。
	作业要求是添加AI单位：
	1.寻敌（在场景中自动搜寻目标）、移动（移动到某个位置或者目标对象）、开火（锁定敌人，进行开火攻击）、躲避（被玩家攻击，逃避到安全位置）
	2.支持黑板变量在节点之间共享数据
	3.基于行为树来组织节点逻辑
	4.实现多种性格的AI，在行为上有一定的差异化，每个行为实现一个单独的行为树1.场景内有透明和半透明效果的材质2.粒子特效3.添加蓝图或C++逻辑驱动材质的变化4.后处理效果
	本次作业主要熟悉了游戏AI相关的知识，包括行为树黑板等模块的使用、AI感知的使用等等。
